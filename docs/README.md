# Echo Documentation

Welcome to the Echo documentation repository. This directory contains the architectural blueprints, technical constraints, and historical decision logs that shape the project.

## Directory Structure

- **[architecture](architecture.md):** The high-level system design, detailing the data flow from the MCP transport layer down to the SQLite persistence engine.
- **[decisions](decisions/README.md):** Architecture Decision Records (ADRs). This is a chronological log of significant engineering trade-offs (e.g., why we use WAL mode, why we migrated to FTS5).
